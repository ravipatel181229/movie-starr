# Movie-Starr

App Created to browse movies/series using themoviedb APIs

## Installation

How to set up

- Create Access token using themoviedb
- setup .net core user secret in format as follows
```json
    {
        "Movies":{
            "TMDBToken": "YOUR API TOKEN"
        }
    }
```
